# Создание справки по системе контроля версий Git #

## Как начать? ##
**Чтобы начать работу в Git, необходимо выполнить следующую команду:**

```
git init
```

## Как добавить файл в отслеживание? ##

```
git add <filename>
```
или
```
git add .
```
